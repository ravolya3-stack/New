# 🍽 Cafe Order Bot

Telegram-бот для заказа блюд из кафе с полным циклом оформления заказа.

## 📁 Структура проекта

```
cafe_bot/
├── bot.py           — Запуск бота
├── config.py        — Настройки (токен, ID группы, название кафе)
├── handlers.py      — Все обработчики сообщений и кнопок
├── keyboards.py     — Клавиатуры и кнопки
├── states.py        — Состояния FSM (конечный автомат)
├── database.py      — Хранилище данных (in-memory)
├── menu_data.py     — Меню кафе (редактируйте под себя)
└── requirements.txt — Зависимости
```

## ⚙️ Установка и запуск

### 1. Установите зависимости
```bash
pip install -r requirements.txt
```

### 2. Настройте config.py
Откройте `config.py` и заполните:
```python
BOT_TOKEN = "ВАШ_ТОКЕН_ОТ_BOTFATHER"
GROUP_ID  = -1001234567890   # ID вашей группы (со знаком минус!)
CAFE_NAME = "Название вашего кафе"
CURRENCY  = "₴"              # или ₽, $, €
```

### 3. Как получить токен бота
1. Напишите @BotFather в Telegram
2. Отправьте `/newbot`
3. Следуйте инструкциям — получите токен

### 4. Как получить ID группы
Способ 1:
1. Добавьте бота `@getidsbot` в вашу группу
2. Он пришлёт ID группы

Способ 2:
1. Откройте `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
2. Перешлите любое сообщение из группы боту
3. Найдите `"chat":{"id": -XXXXXXXXXX}` — это и есть ID

### 5. Добавьте бота в группу
- Добавьте вашего бота в группу кафе как администратора
- Боту нужны права: отправка сообщений

### 6. Запустите бота
```bash
python bot.py
```

## 🍕 Редактирование меню

Откройте `menu_data.py`. Формат:

**Категория БЕЗ подкатегорий:**
```python
"🥗 Салаты": {
    "_has_subcategories": False,
    "items": [
        {"name": "Цезарь", "price": 145, "description": "Курица, сухарики, пармезан"},
    ]
}
```

**Категория С подкатегориями:**
```python
"🍕 Пицца": {
    "_has_subcategories": True,
    "subcategories": {
        "🔴 Классическая": [
            {"name": "Маргарита", "price": 180, "description": "Томат, моцарелла"},
        ],
    }
}
```

## 🔄 Как работает бот

### Для клиента:
1. `/start` — регистрация (имя, телефон, адрес) — только 1 раз
2. **Меню** → Категория → (Подкатегория) → Блюдо → Количество
3. **Корзина** — просмотр с итоговой суммой
4. **Оформить** → Способ оплаты → Подтверждение
5. На экране подтверждения можно изменить телефон, адрес или отправить геолокацию
6. После подтверждения — уведомление о статусе заказа

### Для администратора (в группе):
- Приходит сообщение с полными данными заказа:
  - Имя, телефон, адрес клиента
  - Список блюд с количеством и суммой
  - Итоговая сумма и способ оплаты
- Кнопки: ✅ Принять / ❌ Отклонить
- После нажатия клиент автоматически получает уведомление

## 💾 База данных

По умолчанию используется хранилище в памяти (данные теряются при перезапуске).

Для продакшна рекомендуется подключить SQLite или PostgreSQL — замените функции в `database.py`.

**Рекомендуемые библиотеки:**
- SQLite: `aiosqlite`
- PostgreSQL: `asyncpg` + `SQLAlchemy`

## 📦 Деплой

### На VPS/сервере:
```bash
# Установить screen или tmux
screen -S bot
python bot.py
# Ctrl+A, D — оставить в фоне
```

### С systemd (рекомендуется):
Создайте `/etc/systemd/system/cafebot.service`:
```ini
[Unit]
Description=Cafe Telegram Bot
After=network.target

[Service]
WorkingDirectory=/path/to/cafe_bot
ExecStart=/usr/bin/python3 bot.py
Restart=always

[Install]
WantedBy=multi-user.target
```
```bash
systemctl enable cafebot
systemctl start cafebot
```

### На Railway/Render/Heroku:
Добавьте `Procfile`:
```
worker: python bot.py
```
